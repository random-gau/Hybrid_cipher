# Hybrid Cipher Encryption System

A premium, multi-stage encryption pipeline featuring classic cryptographic algorithms and real-time security analytics.

## Features
- **Multi-Stage Pipeline**: Chains Playfair, Vigenère, and Columnar Transposition ciphers.
- **Key-Strength Predictor**: Real-time evaluation of key complexity and entropy.
- **AI Insights**: Learning component that tracks key patterns and provides relative feedback.
- **Premium UI**: Futuristic "Cyber-Command" dashboard with glassmorphism and smooth animations.
- **Bi-Directional**: Full support for both encryption and decryption pipelines.

## Logic Overview
1. **Playfair**: $5 \times 5$ matrix encryption handling digraphs (I/J mapping).
2. **Vigenère**: Polyalphabetic substitution using a custom keyword.
3. **Columnar**: Geometric transposition based on alphabetical key ordering.

## Running the Application
1. Navigate to the project directory: `cd hybrid_cipher`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open your browser at: `http://localhost:5173`

## Implementation Details
- Built with **React** and **Vite**.
- Custom CSS design system (Vanilla CSS).
- Logic engines implemented in pure JavaScript for performance.
